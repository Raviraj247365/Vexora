Mobile (Flutter)
-----------------

Purpose:
- Cross-platform mobile UI built with Flutter.

Folder role:
- `lib/` for Dart source code
- `android/` and `ios/` for platform integration
- `assets/` for images, icons, sample media

Notes:
- Keep platform-specific code minimal; prefer plugin packages.
- Use `flutter` tooling to manage the app (`flutter pub get`, `flutter run`).
